## Project 1: Inverse Kinematics

This project focuses on inverse kinematics modeling for a robotic manipulator using MATLAB.

### Files
- **Eshun_Lordina_Project1_2D.m**  
  Implements inverse kinematics for a 2D robotic manipulator.

- **Eshun_Lordina_Project1_3D.m**  
  Extends the inverse kinematics formulation to a 3D robotic manipulator.

### Tools Used
- MATLAB

### Key Concepts
- Forward kinematics
- Inverse kinematics
- Position analysis of robotic manipulators
